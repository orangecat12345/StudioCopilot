# StudyCopilot

> 基于国内大语言模型的大学生生成式人工智能辅助学习平台

StudyCopilot 是一个面向高校大学生学习场景设计与开发的生成式人工智能辅助学习平台。项目以国产大语言模型为技术底座，以多智能体协同为核心引擎，围绕大学生在文献阅读、知识理解、课程学习、能力训练与应用创造中的真实需求，构建“搜-学-练”一体化的智能学习闭环。

与通用型对话产品不同，StudyCopilot 并不将 AI 仅作为“答案提供者”，而是尝试将其转化为能够参与知识组织、概念解释、过程引导、能力诊断与学习反馈的学习协作者。平台的整体设计同时吸收了学习科学、认知负荷管理、多模态表征与计算机化自适应测验等理念，目标是在国产模型条件下构建更贴近中国大学生学习语境的 AI 原生学习工作站。

## 项目定位

当前常见生成式 AI 工具在大学生学习场景中普遍存在三类问题：

- 通用工具回答覆盖面广，但缺乏教育场景深耕，容易停留在“给答案”层面。
- 垂类辅导工具更偏向题目求解，难以支持学术阅读、知识建构与高阶思维训练。
- 许多平台侧重单点问答，缺少从资源发现、深度学习到能力训练的完整闭环。

StudyCopilot 的定位是：

- 坚持国产化技术路线，基于智谱 GLM 系列模型构建平台能力。
- 以多智能体协同替代单一模型被动应答，支持复杂学习任务分解与工作流编排。
- 以“搜-学-练”闭环为主线，将信息获取、知识理解与能力训练整合为统一体验。
- 以大学生学习需求为中心，强调深度理解、认知减负、交互引导与个性化支持。

## 核心功能

### 1. StudyCopilot 智能学习工作站

- 支持智能对话、深度思考、联网搜索与文档理解。
- 支持围绕学习材料生成结构化内容，而非仅返回一次性回答。
- 支持基于概念节点的延展式学习，帮助用户形成知识网络。

### 2. AI 课本与沉浸式学习资源生成

- 可根据上传材料自动生成章节化 AI 课本。
- 支持围绕复杂概念生成更具解释性的图文内容与交互式概念卡片。
- 支持模拟课堂、讲稿生成、课件式学习与语音讲解等学习场景。

### 3. 多智能体协同学习交互

- 支持多角色小组讨论、学习陪伴与情境化对话。
- 支持围绕同一学习主题进行多视角解释、回应与协作。
- 将复杂教育任务拆分为不同智能体角色，以提升生成质量与场景适配性。

### 4. 搜索、建构与训练闭环

- 超级搜索模块用于获取和整合学习资料。
- AI 课本、概念解释、闪卡等模块用于知识加工与结构化建构。
- LeetCode CAT 自适应训练、随堂测验等模块用于能力检验与反馈修正。

### 5. AI 应用创造与工作流扩展

- 提供 AI 应用创造控制台与应用市场入口。
- 支持围绕具体学习目标生成垂直工具、工作流或互动内容。
- 将学习平台从单一工具扩展为可持续演化的学习生态。

## 平台特色

### 国产大模型底座

- 全面围绕国内大语言模型能力进行构建。
- 更适配中文语境、国内学习资料与本土化教学场景。
- 有利于兼顾数据安全、技术自主与实际部署需求。

### 多智能体协同机制

- 平台并非依赖单一提示词完成所有任务。
- 不同模块内含面向学习规划、内容生成、诊断反馈、互动表达的专职智能体。
- 通过角色分工与工作流串联，把大模型能力转化为可执行的教学型服务。

### 学习科学导向

- 强调降低信息过载，提升知识建构质量。
- 重视从“给结论”转向“给过程、给路径、给反馈”。
- 将自适应测验、认知负荷管理和多模态学习支持融入系统设计。

### 沉浸式原生交互

- 采用原生 JavaScript + Flask 的轻量架构实现多舞台工作区。
- 支持 SSE 流式输出、沉浸式工作台、全屏学习舞台和动态交互界面。
- 强调学习过程中的持续反馈感，而不是单次静态结果页。

## 典型场景

- 文献阅读与综述理解
- 课程知识梳理与 AI 课本生成
- 概念解释、思维发散与知识点复习
- 编程训练与 LeetCode 自适应刷题
- 多角色讨论、课堂模拟与学习陪伴
- 面向学习需求的 AI 应用快速生成

## 技术架构

### 后端

- `Python`
- `Flask`
- `Server-Sent Events (SSE)` 流式响应
- 多服务模块协作，包括对话、超级搜索、学习日志、LeetCode CAT 与工坊生成链路

### 前端

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla JS)`
- 多舞台工作区、沉浸式界面、动态交互与流式渲染

### 模型与能力

- 智谱 `GLM` 系列大语言模型
- 图像生成能力
- 多模态理解能力
- 联网搜索与内容增强能力

## 主要模块

- `app.py`
  - Flask 应用入口，负责页面路由、API 路由与核心服务编排。
- `ai_service.py`
  - 通用 AI 服务层，负责模型调用、对话生成与部分系统级提示词能力。
- `glm5_agent.py`
  - StudyCopilot 工坊与多类学习智能体逻辑的核心实现文件。
- `super_search_service.py`
  - 超级搜索与资料增强相关服务。
- `journal_service.py`
  - 学习日志、结构规划与图文生成相关服务。
- `leetcode_service.py`
  - LeetCode CAT 自适应训练、能力评估与学习反馈相关服务。
- `templates/`
  - 前端页面模板。
- `static/`
  - 前端脚本、样式与静态资源。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动项目

```bash
python app.py
```

### 3. 打开平台

浏览器访问：

```text
http://127.0.0.1:5000
```

### 4. 配置智谱 API Key

平台支持在运行后通过设置界面填写智谱 API Key，并在不影响正常启动的情况下生效。若未配置有效 Key，部分依赖模型调用的功能将无法正常生成内容。

## 研究与实现要点

本项目对应的研究与实现重点包括：

- 基于国产大语言模型的大学生学习场景适配

- 多智能体协同驱动的学习任务分解与执行
- 面向“搜-学-练”闭环的产品架构设计
- 基于 CAT 的能力诊断与个性化训练推荐
- 面向沉浸式学习体验的原生交互界面设计

## 适用对象

- 高校大学生
- 需要进行深度阅读、知识建构与能力训练的学习者
- 对国产大模型教育应用、AI 学习平台设计与多智能体系统感兴趣的研究者与开发者
